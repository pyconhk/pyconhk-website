每一個宏大計畫，都需要穩固的起點。對 PyCon Hong Kong 這類以社群為本的活動而言，我們的願景來自開發者的熱情，也仰賴那些不只看見未來，而是真正有能力搭建未來基礎的夥伴。今天，我們非常興奮地宣布，將與一位真正的科技巨擘攜手同行。

請一起熱烈歡迎 [Amazon Web Services (AWS)](https://aws.amazon.com/)，正式以 **鑽石贊助** 💎 身分加入 PyCon HK 2025 的航程！

## **數位世界的核心引擎**

近二十年來，AWS 不只是一個服務供應商，更是現代網際網路的重要基石。它是一套平時未必會直接看見、卻力量驚人的基礎設施，支撐著我們日常使用的應用、推動科研上的資料分析突破，也讓新創團隊從第一天起就能以全球市場為目標建立產品。

AWS 提供超過 240 項功能完整的服務，如同一套大型數位 LEGO，讓開發者與團隊自由組合。這股能力背後，是一個分布全球 38 個區域、120 個 Availability Zones 的基礎網絡，為世界各地的創新提供真正全球化的舞台。從智利到沙地阿拉伯，AWS 仍持續擴展新區域，積極將雲端能力帶往更多地方。無數使用者——從車庫創業者到全球大型企業——都依靠 AWS 以更靈活的方式創造未來、放大願景，並探索下一個可能。

## **Python 與 AWS：互相成就的強大組合**

如果 AWS 是雲端世界的架構，那麼 Python 就是其中最通用的語言之一。對我們社群中的許多開發者而言，AWS 並不是遙遠又抽象的概念，而是每天都會使用的工作平台；兩者之間的連結自然且強大。

我們透過 Python 與 `boto3` SDK，用幾行程式碼便能編排龐大而複雜的基礎設施；我們在 AWS Lambda 上撰寫 serverless function，輕鬆應對擴展；我們也在 Amazon SageMaker 上打造 AI 應用，並部署可服務數百萬人的大型 web application。

AWS 對 PyCon Hong Kong 的支持，正是對這種互相成就關係的肯定。這份支持不只是向每一位曾經部署 container、訓練模型或設定資料庫的開發者致意，更是對那些把 AWS 強大工具轉化為具體方案的人所投入的資源。

## **這對我們社群旅程代表什麼？**

AWS 這次重量級的鑽石贊助，將大幅提升我們打造世界級會議體驗的能力。這份支持讓我們可以：

- **策劃更難忘的節目內容**，加入更多前沿 talks、深度 workshops 與具視野的 keynotes。
- **把世界各地最優秀的想法帶到香港**，讓現場成為 ideas 與經驗交流的國際熔爐。
- **全面提升參加者體驗**，提供更優質的資源與更出色的 networking 機會。
- **讓社群連結更緊密**，激發會議後仍能延續的合作火花。

這次合作將使 PyCon HK 2025 不只是一場活動，而是成為本地與區域科技社群的重要里程碑。

## **一起建造下一步未來**

我們衷心感謝 Amazon Web Services (AWS) 對香港 Python 社群的信任與支持。這份支持同時也是一份共同宣言：未來不是等待而來，而是由我們一起建造出來的。

請大家一起向 AWS 致謝，感謝他們成為我們使命中極為重要的基礎夥伴。我們非常期待這次社群聚首時，能擦出更多創新的火花。

一起準備建造下一個未來吧！🚀🐍
