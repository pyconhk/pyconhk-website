## **從第一行** `print("Hello, World")` **到全球雲端**

不論你是否有意識到，只要你是 Python 開發者，你的工作其實早已與 Red Hat 協助建立的世界緊密相連。

想像一下你平時寫 code 嗰條路：

- 你寫下一個 Python script。佢喺邊度跑？好大機會係 **Linux** 伺服器，而 Linux 正正係 Red Hat 多年來推動並打造成企業級現代 IT 基石嘅核心，亦即 **Red Hat Enterprise Linux (RHEL)**。
- 當你要自動化部署、管理設定、編排複雜流程時，你會用上 **Ansible** 呢個強大嘅 automation platform。而 Ansible 係用咩建成？**Python。** 呢個就係一個由社群語言打造、再回到社群手上嘅工具。
- 當你嘅應用需要可擴展、有韌性，而且係 cloud-native，你會將佢打包成 container，再部署去 Kubernetes。Red Hat 嘅 **OpenShift** 正正提供咗業界領先嘅 hybrid cloud platform，令 Python 應用可以由開發者手提電腦一路自然走向全球使用者。

Red Hat 為 Python 呢場變革提供咗工業級引擎室。佢哋建立嘅係路、橋同電網，而我哋嘅程式碼正正就係喺呢啲基礎之上運行。
