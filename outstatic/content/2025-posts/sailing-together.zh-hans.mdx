## TL;DR：PyCon HK 2025

**地点**：香港城市大学杨建文学术楼 LT-13 至 LT-16

**第一日 - Conference Day**：2025 年 10 月 11 日 —— 四条主题路线，涵盖 talks、workshops 与 booths。[立即报名](https://pyconhk2025.eventbrite.com)！

**第二日 - Development Sprint**：2025 年 10 月 12 日 —— 一起为开源项目写 code。[立即报名](https://pyconhk2025.eventbrite.com)，也欢迎[提交项目或报名成为 Sprint Lead](https://docs.google.com/forms/d/e/1FAIpQLSfLAJK00cqFZIvgXyZi-d-UONpIoUo74Jd6w8MXbpmgb8FcoA/viewform)！

PyCon HK 2025 是香港最具代表性的 Python conference 之一，也是全球每年 50 多场 PyCon 活动之一。大会由 [Hong Kong Python User Group](https://www.facebook.com/groups/hkpug)、[Open Source Hong Kong (OSHK)](https://opensource.hk/) 与 [Hong Kong Creative Open Technology Association (HKCOTA)](https://hkcota.org/) 联合举办。今年主题为 *Sailing Together: "Raise the Sail, Let Python Prevail"*。随着 Python 成为推动 AI 创新的核心语言，它同时也是许多人投入程序世界的起点。正如船队并肩航行，我们诚邀开发者、学习者与爱好者一同展开协作与开源成长的旅程。

我们诚邀你于 2025 年 10 月 11 日来到香港城市大学杨建文学术楼 4/F LT-13 至 LT-16，参与充满活力的 Conference Day，现场将有启发性的 talks、动手 workshops 与交流机会。其后在 2025 年 10 月 12 日，还有 Development Sprint 接力登场。一起扬帆启航，让 Python 在香港继续发光！

## 第一天 - Conference Day：2025 年 10 月 11 日

PyCon HK 2025 的 Conference Day 将以四条充满活力的 tracks 带来 talks、动手 workshops 与互动 booths，汇聚香港城市大学内外的 Python 爱好者。今年焦点内容包括：

### [Keynote] Georgi Ker：pip install community

Georgi Ker 是 Python Software Foundation 董事，同时亦是 PyLadies Bangkok 共同创办人。她将分享 Python 全球社群如何成就这门语言今日的成功，也会谈到 Foundation 的角色、本地 meetup 的重要性，以及你可以如何参与和贡献。

### Hemangi Karchalkar：OpenTelemetry Foundations: Hands-On Guide

EPAM Systems 资深软件开发员 Hemangi Karchalkar 将带来一场适合初学者的 OpenTelemetry 工作坊。参加者可由 observability 基础开始，并在 GitHub 资源与教学支援下，用最少代码为应用加入 instrumentation。

### Dr. Adrian Tam：How AI Tools Changed Python

拥有 20 年 Python 经验的 Dr. Adrian Tam，将探讨 GitHub Copilot 等 AI 工具如何改变 Python 开发方式，并分享实例、未来趋势，以及开发者在 AI 时代中可以如何应对。

完整议程可参阅 [https://pycon.hk/zh-hans/schedule](https://pycon.hk/zh-hans/schedule)。

[立即登记门票！](https://pyconhk2025.eventbrite.com)

## 第二天 - Development Sprint：2025 年 10 月 12 日

10 月 12 日的 Development Sprint 是一场协作式 coding 活动，集合 maintainer、contributor、新手与 project leader，在集中时段内一起推进开源项目。无论你是初学者还是资深开发者，这都是你参与 Python 生态圈贡献的好机会。

我们现正招募 Sprint Projects 与 Sprint Leads！欢迎你一起来写 code、学习与交流。
