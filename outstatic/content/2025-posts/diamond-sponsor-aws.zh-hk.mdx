## **數碼世界的核心引擎**

近二十年來，AWS 唔止係一個服務供應商，更係現代互聯網嘅重要基石。佢係一套平時未必會直接見到、但力量非常大嘅基礎設施，支撐住我哋日常使用嘅應用、推動科研上嘅數據分析突破，亦令初創公司由第一日起就可以面向全球市場建立產品。

AWS 提供超過 240 項功能完整嘅服務，好似一套好大型嘅數碼 LEGO，畀開發者同團隊自由組合。呢股能力背後，有一個分布全球 38 個區域、120 個 Availability Zones 嘅基礎網絡支撐，為世界各地嘅創新提供真正全球化嘅舞台。由智利到沙地阿拉伯，AWS 依然持續擴展新區域，積極將雲端能力帶去更多地方。無數用戶——由車房創業者到全球大型企業——都靠 AWS 用更靈活嘅方式創造未來、放大願景，並探索下一個可能。

## **Python 與 AWS：互相成就的強大組合**

如果 AWS 係雲端世界嘅架構，咁 Python 就係其中一門最通用嘅語言。對我哋社群入面好多開發者嚟講，AWS 唔係遙遠又抽象嘅概念，而係每日都會用到嘅工作平台；兩者之間嘅連結自然又實在。

我哋透過 Python 同 `boto3` SDK，用幾行程式碼就可以編排龐大又複雜嘅基礎設施；我哋喺 AWS Lambda 上寫 serverless function，輕鬆應付擴展；我哋亦會喺 Amazon SageMaker 上打造 AI 應用，或者部署可以服務數百萬人嘅大型 web application。

AWS 對 PyCon Hong Kong 嘅支持，正正係對呢種互相成就關係嘅肯定。呢份支持唔止係向每一位曾經部署 container、訓練模型或者設定資料庫嘅開發者致意，更係對一班將 AWS 強大工具變成實際方案嘅人投下資源。
