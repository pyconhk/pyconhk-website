在科技世界里，有些合作一看就知道十分契合。它们不像单纯的商业安排，更像是一群相信相同价值的人再次聚首。今天，我们不只是在宣布一位赞助商，而是在庆祝我们生态圈中的重要一环。

PyCon Hong Kong 非常荣幸欢迎 [Red Hat](https://access.redhat.com/recognition)——全球领先的企业级开源解决方案供应商——成为 2025 年的 **白金赞助**！

## **不只是赞助商，更是一路同行的伙伴**

要理解这次合作为何如此重要，就必须看看 Python 与 Red Hat 的 DNA。对我们而言，Python 不只是一门语言，更是一个建立在协作、开放与知识共享之上的社区。对 Red Hat 而言，open source 也不只是一种模式，而是他们身份本身。多年来，他们持续实践并推动我们社群珍视的原则：协作、透明与 meritocracy。

Red Hat 不只是使用 open source，而是真正活在开源之中。他们公开开发、持续回馈数以千计的项目，也不断培育让 Python 这类技术能在全球企业层面持续成长的生态。这不是由上而下的赞助，而是两个相信同一件事的社群之间的合作：最好的解决方案，从来都是一起建造出来的。

## **从第一行** `print("Hello, World")` **到全球云端**

不论你是否意识到，只要你是 Python 开发者，你的工作其实早已与 Red Hat 协助建立的世界紧密相连。

想象一下你平时写 code 的旅程：

- 你写下一个 Python script。它在哪里执行？很可能是在 **Linux** 服务器上，而 Linux 正是 Red Hat 多年来推动并打造为企业级现代 IT 基石的核心，也就是 **Red Hat Enterprise Linux (RHEL)**。
- 当你需要自动化部署、管理设置、编排复杂流程时，你会用上 **Ansible** 这个强大的 automation platform。而 Ansible 是以什么建成的？**Python。** 这是一个用社群语言打造、再回馈社群的工具。
- 当你的应用需要可扩展、有韧性，而且是 cloud-native 时，你会把它打包成 container，再部署到 Kubernetes。Red Hat 的 **OpenShift** 正正提供了业界领先的 hybrid cloud platform，让 Python 应用可以由开发者手提电脑一路自然走向全球用户。

Red Hat 为 Python 这场变革提供了工业级的引擎室。他们建立的是路、桥与电网，而我们的代码正是行走在这些基础之上。

## **推动我们最有野心的一届 PyCon**

Red Hat 作为白金赞助所带来的支持，就像为 PyCon Hong Kong 2025 注入额外推力。这份支持会直接用在打造一场真正反映社群抱负与创意的会议，让我们可以：

- **策划世界级节目内容**，邀请活跃于 Python、AI、cloud 与 automation 交汇处的先行者与专家。
- **降低参与门槛**，让学生、转职者与不同背景的开发者都更容易参与。
- **创造无可比拟的 networking 空间**，让一次普通对话也有机会成为下一个大型开源项目或改变职业生涯的起点。
- **投资高质量 hybrid 体验**，让开放知识分享的精神触及更多人、更多地方。

## **公开地，一起建造未来**

我们衷心感谢 Red Hat 的支持。这次合作清楚肯定了 Python 社群在推动未来科技中的角色，也代表一份共同承诺：创新应该是开放、协作，而且人人都可以参与。

请大家一起向 Red Hat 致上谢意，感谢他们成为开源世界中值得信赖的伙伴。我们已经非常期待看到，这次聚首会带来怎样的成果。

一起在 PyCon HK 2025 编码、协作、创造吧！🐧🐍
