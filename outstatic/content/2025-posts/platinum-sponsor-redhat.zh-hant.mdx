在科技世界裡，有些合作一看就知道十分契合。它們不像單純的商業安排，更像是一群相信相同價值的人再次聚首。今天，我們不只是在宣布一位贊助商，而是在慶祝我們生態圈中的重要一環。

PyCon Hong Kong 非常榮幸歡迎 [Red Hat](https://access.redhat.com/recognition)——全球領先的企業級開源解決方案供應商——成為 2025 年的 **白金贊助**！

## **不只是贊助商，更是一路同行的夥伴**

要理解這次合作為何如此重要，就必須看看 Python 與 Red Hat 的 DNA。對我們而言，Python 不只是一門語言，更是一個建立在協作、開放與知識共享之上的社群。對 Red Hat 而言，open source 也不只是一種模式，而是他們身份本身。多年來，他們持續實踐並推動我們社群珍視的原則：協作、透明與 meritocracy。

Red Hat 不只是使用 open source，而是真正活在開源之中。他們公開開發、持續回饋數以千計的專案，也不斷培育讓 Python 這類技術能在全球企業層面持續成長的生態。這不是由上而下的贊助，而是兩個相信同一件事的社群之間的合作：最好的解決方案，從來都是一起建造出來的。

## **從第一行** `print("Hello, World")` **到全球雲端**

不論你是否意識到，只要你是 Python 開發者，你的工作其實早已與 Red Hat 協助建立的世界緊密相連。

想像一下你平時寫 code 的旅程：

- 你寫下一個 Python script。它在哪裡執行？很可能是在 **Linux** 伺服器上，而 Linux 正是 Red Hat 多年來推動並打造成企業級現代 IT 基石的核心，也就是 **Red Hat Enterprise Linux (RHEL)**。
- 當你需要自動化部署、管理設定、編排複雜流程時，你會用上 **Ansible** 這個強大的 automation platform。而 Ansible 是以什麼建成的？**Python。** 這是一個用社群語言打造、再回饋社群的工具。
- 當你的應用需要可擴展、有韌性，而且是 cloud-native 時，你會把它打包成 container，再部署到 Kubernetes。Red Hat 的 **OpenShift** 正正提供了業界領先的 hybrid cloud platform，讓 Python 應用可以由開發者手提電腦一路自然走向全球使用者。

Red Hat 為 Python 這場變革提供了工業級的引擎室。他們建立的是路、橋與電網，而我們的程式碼正是行走在這些基礎之上。

## **推動我們最有野心的一屆 PyCon**

Red Hat 作為白金贊助所帶來的支持，就像為 PyCon Hong Kong 2025 注入額外推力。這份支持會直接用在打造一場真正反映社群抱負與創意的會議，讓我們可以：

- **策劃世界級節目內容**，邀請活躍於 Python、AI、cloud 與 automation 交會處的先行者與專家。
- **降低參與門檻**，讓學生、轉職者與不同背景的開發者都更容易參與。
- **創造無可比擬的 networking 空間**，讓一次普通對話也有機會成為下一個大型開源專案或改變職涯的起點。
- **投資高品質 hybrid 體驗**，讓開放知識分享的精神觸及更多人、更多地方。

## **公開地，一起建造未來**

我們衷心感謝 Red Hat 的支持。這次合作清楚肯定了 Python 社群在推動未來科技中的角色，也代表一份共同承諾：創新應該是開放、協作，而且人人都可以參與。

請大家一起向 Red Hat 致上謝意，感謝他們成為開源世界中值得信賴的夥伴。我們已經非常期待看到，這次聚首會帶來怎樣的成果。

一起在 PyCon HK 2025 編碼、協作、創造吧！🐧🐍
