大きな挑戦には、それを押し上げる発射台が必要です。PyCon Hong Kong のようなコミュニティ主導イベントにとって、その原動力は開発者たちの情熱と、未来を見据えるだけでなく実際にその土台を築くパートナーです。今日は、そんな力強い協力関係を発表できることを心からうれしく思います。

[Amazon Web Services (AWS)](https://aws.amazon.com/) が、PyCon HK 2025 の **Diamond Sponsor** 💎 として参加してくださることになりました。どうぞ温かくお迎えください！

## **デジタル世界を支える中枢**

AWS はこの 20 年近く、単なるサービス提供者ではなく、現代インターネットの基盤そのものであり続けてきました。私たちが日々使うアプリを配信し、科学的ブレークスルーのためのデータを処理し、創業間もないスタートアップが初日から世界を目指すことを可能にする、目に見えないけれど非常に強力な存在です。

240 以上のフル機能サービスを擁する AWS は、開発者にとって究極のデジタル LEGO セットのような存在です。世界 38 リージョン・120 アベイラビリティーゾーンにまたがるグローバルネットワークにより、イノベーションの舞台を文字通り世界規模で支えています。チリからサウジアラビアに至るまで新リージョンの展開も続いており、クラウドをあらゆる場所へ届けるという使命に対する AWS の本気度がうかがえます。ガレージの中の個人開発者から世界最大級の企業まで、数百万のユーザーが AWS 上で俊敏に構築し、大きく夢を描き、次の標準をつくっています。

## **Python と AWS: 切っても切れない関係**

もし AWS がクラウドの設計図だとすれば、Python はその共通言語です。コミュニティの多くの開発者にとって、AWS は遠い存在ではなく、日々の仕事場そのものです。このつながりはごく自然で、とても強力です。

私たちは Python と `boto3` SDK を使い、わずかなコードで大規模かつ複雑なインフラを操作します。AWS Lambda 上では、美しく力強いサーバーレス関数を書き、負荷が増えても軽やかにスケールさせます。Amazon SageMaker では AI の未来を形にし、数百万人に届く高度な Web アプリケーションをデプロイします。

AWS が PyCon Hong Kong を支援してくださることは、この相互関係への深い理解と敬意の表れです。コンテナをデプロイし、モデルを学習させ、データベースを構成してきたすべての開発者へのエールでもあります。AWS の強力なツール群を、実際の価値あるソリューションへ変えている人たちへの投資なのです。

## **この支援がコミュニティにもたらすもの**

AWS によるダイヤモンドスポンサーシップは、PyCon Hong Kong 2025 をさらに前へ進める強力な推進力です。この支援によって、私たちは次のことを実現できます。

- **忘れられないプログラムを編成すること**。より先進的なトーク、深掘りワークショップ、ビジョナリーなキーノートを届けられます。
- **世界中の優れた知見を香港へ招くこと**。アイデアと専門性が交差する、国際色豊かな場をつくれます。
- **参加者体験をさらに高めること**。充実した環境と、質の高いネットワーキング機会を提供できます。
- **コミュニティのつながりをより強くすること**。カンファレンス後も続く協力関係のきっかけを生み出せます。

このパートナーシップによって、PyCon HK 2025 は単なるイベントではなく、この地域の技術コミュニティにとって節目となる場になるはずです。

PyCon HK のミッションを支える土台となってくださる AWS に、ぜひ皆さんも感謝をお送りください。このコミュニティが集まったとき、どんな火花が生まれるのか、私たちもとても楽しみにしています。
