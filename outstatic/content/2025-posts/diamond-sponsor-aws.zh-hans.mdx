请一起热烈欢迎 [Amazon Web Services (AWS)](https://aws.amazon.com/)，正式以 **钻石赞助** 💎 身分加入 PyCon HK 2025 的航程！

## **数字世界的核心引擎**

近二十年来，AWS 不只是一家服务供应商，更是现代互联网的重要基石。它是一套平时未必会直接看见、却力量惊人的基础设施，支撑着我们日常使用的应用、推动科研上的数据分析突破，也让新创团队从第一天起就能以全球市场为目标建立产品。

AWS 提供超过 240 项功能完整的服务，如同一套大型数字 LEGO，让开发者与团队自由组合。这股能力背后，是一个分布全球 38 个区域、120 个 Availability Zones 的基础网络，为世界各地的创新提供真正全球化的舞台。从智利到沙地阿拉伯，AWS 仍持续扩展新区，积极将云端能力带往更多地方。无数用户——从车库创业者到全球大型企业——都依靠 AWS 以更灵活的方式创造未来、放大愿景，并探索下一个可能。

## **Python 与 AWS：互相成就的强大组合**

如果 AWS 是云端世界的架构，那么 Python 就是其中最通用的语言之一。对我们社群中的许多开发者而言，AWS 并不是遥远又抽象的概念，而是每天都会使用的工作平台；两者之间的连接自然且强大。

我们透过 Python 与 `boto3` SDK，用几行代码便能编排庞大而复杂的基础设施；我们在 AWS Lambda 上撰写 serverless function，轻松应对扩展；我们也在 Amazon SageMaker 上打造 AI 应用，并部署可服务数百万人的大型 web application。

AWS 对 PyCon Hong Kong 的支持，正是对这种互相成就关系的肯定。这份支持不只是向每一位曾经部署 container、训练模型或设置数据库的开发者致意，更是对那些把 AWS 强大工具转化为具体方案的人所投入的资源。

## **这对我们社群旅程代表什么？**

AWS 这次重量级的钻石赞助，将大幅提升我们打造世界级会议体验的能力。这份支持让我们可以：

- **策划更难忘的节目内容**，加入更多前沿 talks、深度 workshops 与具视野的 keynotes。
- **把世界各地最优秀的想法带到香港**，让现场成为 ideas 与经验交流的国际熔炉。
- **全面提升参加者体验**，提供更优质的资源与更出色的 networking 机会。
- **让社群连接更紧密**，激发会议后仍能延续的合作火花。

这次合作将使 PyCon HK 2025 不只是一场活动，而是成为本地与区域科技社群的重要里程碑。
